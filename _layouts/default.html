Setup Instructions

Follow these steps to optimize your Smite performance:

üóëÔ∏è Clear Your Shader Cache (NVIDIA)
  1. Close all games and GPU-intensive applications
  2. Press Windows + R to open Run dialog
  3. Type %temp% and press Enter
  4. Delete all files in the Temp folder (select all with Ctrl+A, then Delete)
  5. Navigate to C:\ProgramData\NVIDIA Corporation\NV_Cache
  6. Delete all files in the NV_Cache folder
  7. Alternatively, open NVIDIA Control Panel ‚Üí Manage 3D Settings ‚Üí Shader Cache Size ‚Üí Set to your preference or disable/re-enable
üóëÔ∏è Clear Your Shader Cache (AMD)
  1. Close all games and GPU-intensive applications
  2. Press Windows + R to open Run dialog
  3. Type %localappdata% and press Enter
  4. Navigate to AMD\DxCache and AMD\GLCache
  5. Delete all files in both cache folders
  6. Alternatively, open AMD Software ‚Üí Gaming ‚Üí Graphics ‚Üí Advanced ‚Üí Reset Shader Cache
‚ö° Enable Resizable BAR in BIOS

Requirements: Compatible CPU, motherboard, GPU, and updated BIOS

  1. Restart your PC and enter BIOS (usually press Del, F2, or F12 during boot)
  2. Look for "Resizable BAR" or "Re-Size BAR Support" or "Smart Access Memory" (AMD)
  3. Common locations:
    • Advanced ‚Üí PCI Subsystem Settings
    • Advanced ‚Üí System Agent Configuration
    • Settings ‚Üí Advanced ‚Üí PCI Configuration
  4. Enable "Resizable BAR" or "Above 4G Decoding"
  5. You may also need to enable "CSM" as Disabled
  6. Save changes and exit BIOS (usually F10)
  7. Verify it's enabled: Download GPU-Z and check under "Advanced" tab
üöÄ Overclock Your RAM (Enable XMP/DOCP)
  1. Restart your PC and enter BIOS
  2. Look for "XMP" (Intel) or "DOCP/EXPO" (AMD)
  3. Common locations:
    • AI Tweaker ‚Üí Memory Settings (ASUS)
    • Extreme Memory Profile (MSI)
    • Advanced Memory Settings (Gigabyte)
  4. Enable XMP/DOCP and select Profile 1 (usually the highest speed)
  5. Save and exit BIOS (F10)
  6. Verify: Download CPU-Z, check "Memory" tab for actual speed (multiply DDR by 2 for effective speed)
  7. Run a stress test to ensure stability (Windows Memory Diagnostic or MemTest86)
‚öôÔ∏è Overclock Your CPU (If Not X3D)

Warning: Only for non-X3D CPUs. X3D chips like 7800X3D, 7950X3D are already optimized.

Easy Method - Auto OC in BIOS:

  1. Enter BIOS
  2. Look for "AI Overclocking", "Game Boost", or "Precision Boost Overdrive" (AMD)
  3. Enable the automatic overclocking feature
  4. Save and exit

AMD Ryzen - PBO Method:

  1. Enter BIOS ‚Üí Advanced ‚Üí AMD Overclocking
  2. Enable "Precision Boost Overdrive"
  3. Set "PBO Limits" to "Motherboard"
  4. Adjust "Curve Optimizer" to negative values (-5 to -30 per core, test for stability)
  5. Save and stress test with Cinebench or Prime95

Intel - Easy OC:

  1. Enter BIOS ‚Üí Overclocking or AI Tweaker
  2. Increase "CPU Core Ratio" by 1-2 (e.g., 50 for 5.0 GHz)
  3. May need to increase voltage slightly (start with Auto or +0.05V)
  4. Save and test with Cinebench or Prime95
  5. If unstable, reduce ratio or increase voltage slightly
