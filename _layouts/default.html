Smite Optimization Settings Guide

Warning: This guide includes BIOS modifications. Incorrect settings can prevent your PC from booting. Research your specific hardware and proceed at your own risk.

Performance Expectations

Smite has awful optimization. You can't get decent frame rates with better graphics settings no matter how powerful your hardware is. Even with a maxed out PC like mine (see here) running an RTX 5090 - which would destroy any other game - you'll still only hit 200-250 FPS with these settings. The game just doesn't perform like it should.

  3. Delete all files in the NV_Cache folder

üóëÔ∏è Clear Your Shader Cache (AMD)

Note: Close Smite and all performance-intensive applications before clearing shader cache.

  1. Press Windows + R to open Run dialog
  2. Type %localappdata% and press Enter
  3. Navigate to AMD\DxCache and delete all files
  4. Navigate to AMD\GLCache and delete all files

Alternative: Open AMD Software ‚Üí Gaming ‚Üí Graphics ‚Üí Advanced ‚Üí Reset Shader Cache

‚ö° Enable Resizable BAR in BIOS

‚ö†Ô∏è Warning: BIOS settings vary by motherboard. Look up your specific motherboard model for exact instructions.

  1. Restart your PC and press Del, F2, or F12 during boot to enter BIOS
  2. Find "Resizable BAR" (may also be called "Re-Size BAR Support" or "Smart Access Memory")
  3. Enable it
  4. You might need to enable "Above 4G Decoding" as well - look up a guide for your motherboard
  5. Save and exit (F10)
  6. Verify: Check in BIOS or the GPU software to confirm
üöÄ Enable RAM XMP/DOCP Profile

‚ö†Ô∏è Warning: BIOS settings vary by motherboard. Look up your specific motherboard model for exact instructions.

  1. Restart and enter BIOS
  2. Find "XMP" (Intel) or "DOCP"/"EXPO" (AMD) in memory settings
  3. Enable it and select Profile 1
  4. Save and exit (F10)
  5. Verify: Task Manager ‚Üí Performance ‚Üí Memory ‚Üí Check the speed
‚öôÔ∏è Enable CPU Auto Overclocking (If Not X3D)

‚ö†Ô∏è Warning: BIOS settings vary by motherboard. Look up your specific motherboard model for exact instructions. X3D chips (7800X3D, 7950X3D, 9800X3D) should NOT be overclocked.

  1. Enter BIOS
  2. Find your motherboard's auto overclocking feature (varies by brand)
  3. Enable it
  4. Save and exit (F10)

For AMD: Enable "Precision Boost Overdrive (PBO)"

Check speeds: Task Manager ‚Üí Performance ‚Üí CPU
