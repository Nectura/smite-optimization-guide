SMITE 2 Optimization Guide

Warning: This guide includes BIOS modifications. Incorrect settings can prevent your PC from booting. Research your specific hardware and proceed at your own risk.

Performance Expectations

Smite has awful optimization. You can't get decent frame rates with better graphics settings no matter how powerful your hardware is. Even with an essentially maxed out PC like mine (see here) - which would destroy any other game - you'll still only hit 200-250 FPS with these settings. The game just doesn't perform like it should.

üíæ Update Your Motherboard BIOS

‚ö†Ô∏è Important Warnings:

  • Only update if you have a specific reason (new CPU support, bug fixes, security patches, required missing features, or if your BIOS is extremely outdated and hasn't been updated in several years)
  • Power loss during update can brick your motherboard - use a UPS (uninterruptible power supply) if possible
  • Installing the wrong BIOS file can permanently damage your motherboard
  • BIOS updates may reset all custom settings (overclocks, XMP, boot order) - write them down first

Step 1: Check Your Current BIOS Version

  1. Press Windows + R, type msinfo32, and press Enter
  2. Look for "BIOS Version/Date" in the System Summary section
  3. Write down your current version number

Step 2: Download the Correct BIOS Update

  1. Find your exact motherboard model (check System Information or physically look at the board)
  2. Go to your motherboard manufacturer's official website (ASUS, MSI, Gigabyte, ASRock, etc.)
  3. Navigate to Support ‚Üí Downloads for your specific model
  4. Download the latest non-beta BIOS version
  5. Triple-check the model number matches exactly - wrong BIOS = bricked motherboard

Step 3: Prepare USB Drive

  1. Use a USB drive (doesn't need to be high quality)
  2. Format it as FAT32 (right-click drive in File Explorer ‚Üí Format ‚Üí select FAT32)
  3. Extract the downloaded BIOS files (usually a .zip file)
  4. Copy the extracted BIOS file to the USB drive root directory

Step 4: Update the BIOS

Most modern motherboards use this method. If yours doesn't, look up a guide for your specific model.

  1. Restart your PC and press Del, F2, or F12 to enter BIOS
  2. Find the BIOS update utility (called "EZ Flash," "M-Flash," "Q-Flash," or similar)
  3. Select the BIOS file from your USB drive
  4. Confirm and start the update

DO NOT turn off or restart your PC during the update (5-10 minutes)

The system will reboot automatically when the update is complete

Step 5: After Update

  1. Verify the new BIOS version in System Information or BIOS screen
  2. Reconfigure any custom settings you wrote down earlier
  3. Test system stability (boot into Windows, run a few programs, and ensure there are no crashes or errors)

If Something Goes Wrong:

  • Try clearing CMOS (remove motherboard battery for 30 seconds or use reset button)
  • Some motherboards have dual BIOS that can auto-recover
  • Contact manufacturer support if all else fails
‚ö° Enable Resizable BAR in BIOS

‚ö†Ô∏è Important Requirements:

  • Your CPU, motherboard, and GPU must all support Resizable BAR
  • Your motherboard BIOS must be updated to a version that supports Resizable BAR - older BIOS versions may not have this option available
  • BIOS settings vary by motherboard - look up your specific model for exact instructions
  • Your system must be using UEFI boot mode (not Legacy/CSM)
  • Your drive must use GPT partition (not MBR)

Steps:

  1. Restart your PC and press Del, F2, or F12 during boot to enter BIOS
  2. Disable "CSM" (Compatibility Support Module) or "Legacy Boot"
  3. Enable "Above 4G Decoding"
  4. Enable "Resizable BAR" (may also be called "Re-Size BAR Support" or "Smart Access Memory" for AMD)
  5. Save and exit (usually F10)
  6. Verify: Use GPU-Z (Advanced ‚Üí Resizable BAR) to confirm it's enabled

Note: Some older GTX 16 series and RTX 30 series GPUs may also need a VBIOS update. Check your GPU manufacturer's website.

üöÄ Enable RAM XMP/DOCP Profile

‚ö†Ô∏è Warning: BIOS settings vary by motherboard. Look up your specific motherboard model for exact instructions.

  1. Restart and enter BIOS
  2. Find "XMP" (Intel) or "DOCP"/"EXPO" (AMD) in memory settings
  3. Enable it and select Profile 1
  4. Save and exit (F10)
  5. Verify: Task Manager ‚Üí Performance ‚Üí Memory ‚Üí Check the speed
‚öôÔ∏è Enable CPU Auto Overclocking (If Not X3D)

‚ö†Ô∏è Warning: BIOS settings vary by motherboard. Look up your specific motherboard model for exact instructions. X3D chips (7800X3D, 7950X3D, 9800X3D) should NOT be overclocked.

  1. Enter BIOS
  2. Find your motherboard's auto overclocking feature (varies by brand)
  3. Enable it
  4. Save and exit (F10)

For AMD: Enable "Precision Boost Overdrive (PBO)"

Check speeds: Task Manager ‚Üí Performance ‚Üí CPU

  1. Visit ExitLag (referral link) and create an account

  5. Click "Apply routes" before launching Smite
  6. Launch Smite and enjoy lower, more stable ping

üõ°Ô∏è Disable Windows Automatic Driver Updates

This prevents Windows from overriding and breaking your graphics drivers. GPU drivers should only be updated through your GPU software (NVIDIA GeForce Experience or AMD Software).

  1. Press Windows + R and type sysdm.cpl
  2. Go to "Hardware" tab ‚Üí "Device Installation Settings"
  3. Select "No (your device might not work as expected)"
  4. Click "Save Changes"
üßπ Remove Unnecessary Software

Some pre-installed manufacturer utilities and third-party programs can consume resources, cause background CPU/GPU load, or interfere with games.

  1. Uninstall bloatware like Armory Crate, MyASUS, HP Command Center, etc.
  2. Remove unnecessary antivirus programs. Modern Windows Defender is sufficient in current year.
  3. Restart your PC after uninstalling to ensure changes take effect.

Note: Focus on software that runs constantly in the background and isn't required for your daily use.
